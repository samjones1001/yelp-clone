## Usage Instructions
* clone the repo and run bundle
```shell
$ git clone https://github.com/samjones1001/makersbnb
$ cd makersbnb
$ rvm 2.3.3
$ bundle
```
* run tests
```shell
$ rspec
```
* start up sinatra
```shell
$ ruby app.rb
```
* in your browser, navigate to http://localhost:4567/
